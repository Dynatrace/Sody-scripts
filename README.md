> This product is not officially supported by Dynatrace!

These are script I have written mostly in dealing with my own Managed Cluster for testing.  If they are also useful to others, I want them to be available

Please note there is typically minimal error checking in the scripts (at least at the start) as they are usually originally targetted to my own testing lab.

Each script or collection of scripts will have its own subdirector and its own README.md describing it

Any script that later gets incorporated into the product gets obsoleted here, and future issues for an obsoleted script will typically only get a recommendation to upgrade to a version that has incorported the functionality.
