This script was created to visually simulate cluster data distribution and
triplicate replication using readable text, and then recovery from multiple
nodes lost.

This was done mostly for fun, but is also useful for illustration purposes.

The included example text is a paragraph from Shakespeare's "Romeo and Juliet"
